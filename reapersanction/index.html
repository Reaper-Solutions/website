Minecraft Plugin ReaperSanction

  • Gui punishment plugin for spigot minecraft server.
  • From 1.8 to latest minecraft versions.

    • FAQ

  • Do I need a sanction minecraft plugin like advancedban or liteban etc ? NO
  • Do I need a database ? Not for the moment, in future updates, you will be able to choose your storage preferences.
  • What do we do if there is a bug ? If you have any bug, please make an issue on github. If you have an issue with the config, delete the config and restart your server.
  • BungeeCord / Velocity proxy version ? Coming soon

    • Installation

  • Put this plugin in your plugins folder
  • Restart or reload your server
  • Change something in config.yml if it's needed
  • Done :)

    • In developpement

  • SQL bans, mutes with a rework of storage

  • Proxy (Velocity/BungeeCord) version
