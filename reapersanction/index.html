Minecraft ReaperSanction

This is a gui punishment plugin for spigot minecraft server. From 1.8 to latest minecraft versions.

Informations

This plugin is a powered gui punishment plugin. If you have any bug, please make an issue on github. If you have an issue with the config, delete the config and restart your server.

FAQ

  • Do I need a sanction minecraft plugin like advancedban or liteban etc ? NO
  • Do I need a database ? No for the moment, in future updates, you will be able to choose your storage preferences.
  • BungeeCord / Velocity proxy version ? Coming soon

    • Installation

  • Put this plugin in your plugins folder
  • Restart or reload your server
  • Change something in config.yml if it's needed
  • Done :)

    • Commands

    Basic moderation commands :

  • /ban /tempban /ban-ip /unban /unmute /rs
  • Permissions: reapersanction.ban, reapersanction.tempban, reapersanction.banip, reapersanction.unban, reapersanction.unmute, reapersanction

    /vanish or /v

  • Permission: reapersanction.vanish
  • Hide you from other people

    /rsadmin < rl or reload or infos>

  • Permission: reapersanction.rsadmin
  • rl or Reload : Reload the config
  • infos : Show my discord # and my email

    /report < player-name >

  • Permission: no permission needed
  • Report someone with default reason in config
  • The reported player doesn't need to be online
  • If you want to see report message you need the permission "reportview"
  • This command can be toggle !
