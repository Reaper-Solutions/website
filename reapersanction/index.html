Minecraft ReaperSanction

This is a gui punishment plugin for spigot minecraft server. From 1.8 to latest minecraft versions.

Download

Latest version

Informations

This plugin is a powered gui punishment plugin. If you have any bug, please make an issue on github. If you have an issue with the config, delete the config and restart your server.

FAQ

  • Do I need a sanction minecraft plugin like advancedban or liteban etc ? NO
  • Do I need a database ? No for the moment, in future updates, you will be able to choose your storage preferences.
  • BungeeCord / Velocity proxy version ? Coming soon

    • Installation

  • Put this plugin in your plugins folder
  • Restart or reload your server
  • Change something in config.yml if it's needed
  • Done :)

    • In developpement

  • SQL bans, mutes with a rework of storage

    Commands, permissions & more

    Commands Alias Sub commands Args Action Permission
    /rs /reapersanction player_name Open the gui reapersanction
    /ban player_name reason Ban a player reapersanction.ban
    /tempban player_name reason time Tempban a player reapersanction.tempban
    /kick player_name reason Kick a player reapersanction.kick
    /mute player_name reason Mute a player reapersanction.mute
    /tempmute player_name reason time Tempmute a player reapersanction.tempmute
    /ban-ip player_name reason Ban an ip reapersanction.banip
    /vanish /v player_name Vanish a player reapersanction.vanish
    /unban player_name Unban a player reapersanction.unban
    /unmute player_name Unmute a player reapersanction.unmute
    /report player_name Report a player
    /rsadmin rl, reload, infos Reload the config, get the plugin infos reapersanction.rsadmin
